# Hasta-Shilpa: An AI-Powered Design Bridge for Artisans 🎋

Hasta-Shilpa (Handicraft-Craft) is a socially conscious Android application built to empower bamboo and cane artisans. The project utilizes Generative AI to provide modern design innovations, helping traditional craftsmen stay relevant in today’s competitive urban markets.

## 🚀 Features

- **GenAI Design Library:** A curated feed of modern bamboo/cane products (e.g., laptop stands, minimalist lamps) to inspire artisans.
- **Blueprint Viewer:** High-resolution, zoomable technical sketches with precise measurements for production accuracy.
- **Smart Price Suggester:** A logic-based calculator that factors in local material costs and labor hours to recommend fair market prices.
- **Material Tracker:** A simple inventory ledger for artisans to log raw material usage and manage stock.
- **Secure Authentication:** Easy and secure login using Firebase Phone Authentication.

## 🛠️ Tech Stack

- **Language:** [Kotlin](https://kotlinlang.org/)
- **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose) (Material 3)
- **Backend:** [Firebase Authentication](https://firebase.google.com/docs/auth), [Firestore](https://firebase.google.com/docs/firestore)
- **Cloud Storage:** [Firebase Storage](https://firebase.google.com/docs/storage) (for Blueprints & Images)
- **Architecture:** MVVM (Model-View-ViewModel)
- **AI Integration:** Gemini 3 Flash API
